# Commands
https://hexo.io/docs/commands.html
- new: `hexo new [layout] <title>`
  - layout: post, page, draft
  - title: string with quotation marks ('')
- publish(for draft): `hexo publish [layout] <filename>`
- generate: `hexo g`
- server: `hexo s`
- deploy: `hexo d` ( `-g` )
- clean: `hexo clean`
