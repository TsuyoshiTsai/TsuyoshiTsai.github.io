Git #01 準備開始

發表於 2016-12-29

Git 的優點

  • 不依賴網路，能夠在本地端進行版控
  • 備份容易，每個人都擁有完整的開發紀錄
  • 分支的操作相當彈性且輕量，每個分支只是某個 Commit 物件的參考指標而已

學習方法

Git 的指令跟參數非常多，若每次要用 Git 指令時都要查書的話，會非常沒效率，故學習 Git 最終仍需 GUI 工具作為輔助，但一開始還是建議先透過下指令的方式學習 Git 的基礎觀念。

步驟
  1. 了解運作原理
  2. 建立思維模式 ( Mindset )
  3. 尋找適合的相關工具 ( GUI )
重點觀念
  • 學習建立、操作分支 ( Branch )
  • 學習合併檔案 ( Merge )
  • 學習解決衝突 ( Conflict )
