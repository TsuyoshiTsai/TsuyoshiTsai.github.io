### 參照操作
 - `git update-ref [RefName] [Object]`
    - 以 RefName 建立 Object 的**一般參照**
    - 使用時建議使用 refs/[RefName]


 - `git update-ref -d [RefName]`
    - 刪除 RefName 一般參照


 - `git rev-parse [RefName or RelativeName]`
    - 查詢 **參照名稱** 或 **相對名稱** 的絕對名稱
